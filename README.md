# django-ecommerce-project
# Django E-Commerce Backend REST API 🚀

## About

This project is a Django REST Framework based E-Commerce Backend developed as part of my Django learning journey.

It demonstrates backend development concepts including authentication, product management, shopping cart, order management, PostgreSQL database integration, and REST APIs.

---

## Features

- User Registration
- JWT Authentication
- Product API
- Category API
- Product Details API
- Shopping Cart
- Add to Cart
- Update Cart Quantity
- Remove from Cart
- Order Creation
- Order Items
- User Profile
- PostgreSQL Database
- Django ORM
- Django REST Framework
- Media/Image Upload
- JSON API Responses
